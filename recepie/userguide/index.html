A) Overview - top

Recipes Finder - Ionic Full Application V.1.0 is an application built using ionic and phonegap mobile development framework that where greatly assist you in building applications recipes using web technologies that you know and love HTML, CSS, and javascript. You can easily build applications recipes that can be used on a variety of mobile devices such as iOS, Android, Blackberry and other devices in no time only with knowledge of HTML, CSS, and javascript.

Before we start, lets take a look on the folder structure of the app. All app files are placed in the RECIPES_FINDER folder. The RECIPES_FINDER folder consists of 7 more folders for diferent types of app assets:


  1. css - containing all css files, for the theme specific CSS.
  2. db - Sample data & images content for app.
  3. img - containing all images files.
  4. js - containing all javascript files.
  5. lib - containing all library files.
  6. res - containing all splash & icons files.
  7. views - containing all views files.

B) Installation - top

1. Manage content (Sqlite database) - top

  • You can open files recipes.xlsx in the folder db and insert content in accordance with a description of each column and sheets provided


  • The images column you fill the image file name and put the image file in the folder db/img

2. Configuration Files - top

  • For setting or configuration (Key adsense & rating app) of your application you can open the file app.js in folder JS

  • How setting AdMob plugins you can check this link http://ngcordova.com/docs/plugins/adMob
  • How setting AppRate plugins you can check this link http://ngcordova.com/docs/plugins/appRate

4. Compile Application to Native APP - top

  • You can use PhoneGap Build is a cloud-based service built on top of the PhoneGap framework. It allows you to easily build those same mobile apps in the cloud, you can check this link https://build.phonegap.com
